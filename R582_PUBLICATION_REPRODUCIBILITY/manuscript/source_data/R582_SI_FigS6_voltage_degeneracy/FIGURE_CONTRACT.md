# R582 Supplementary Figure S6 contract

**Core conclusion.** Multiple reduced calibration-controlled accessibility
factor/effective-coefficient pairs reproduce the selected voltage contribution,
so the full-cell voltage trace does not uniquely identify the native
positive-electrode remaining-area relation.

**Archetype.** Asymmetric quantitative grid with one hero panel.

**Backend and output.** Python/matplotlib exclusively; exactly 180 mm by 92 mm.
Editable SVG and PDF plus 600 dpi PNG/TIFF and 300 dpi actual-size colour and
grayscale previews. All text uses the exact TeX Gyre Termes OTF faces used by
the `tgtermes` manuscript body; the minimum text size is 6.5 pt.

**Panel map.**

- **a:** the selected existing full-cell charge trace and the reduced
  observation-layer fit over the prespecified fitted interval, with the aligned
  fit-minus-data residual below.
- **b, hero:** the reduced-factor/effective-coefficient relation for a fixed
  70.813 mV selected voltage contribution. The horizontal variable is
  $R_{\theta,\mathrm{end}}=1-\theta_{\mathrm{end}}$; registered parameter
  pairs are shown as open circles.

**Evidence hierarchy.** Panel b carries the identifiability result. Panel a
shows exactly which trace, fit interval and residual support that diagnostic.

**Statistics.** The fit is deterministic and uses one selected charge trace;
no independent replicate statistic or inferential error bar applies. RMSE is a
fit residual summary, not experimental uncertainty.

**Source data.** Four byte-frozen registered R532/R538 files are copied under
`inputs/` and verified by SHA-256 before use. Historical input column names are
preserved byte-for-byte, while clean R582 plotted-source tables use
`R_theta_end`. Parameter tables are regenerated by the script.

**Reviewer risk and boundary.** $R_{\theta}$ is the reduced,
calibration-controlled accessibility factor. It is **not** the native remaining
area: $A_{\mathrm{bare}}/A_0=R_{\theta}T_{\mathrm{pore}}$. The registered
reduced-factor trajectory supplies the fit shape. This is not independent
electrode-resolved validation, and no point or curve is interpreted as a
deposit morphology.
